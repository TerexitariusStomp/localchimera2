What you're installing:

How it works:

  • When idle, it automatically switches to mining across multiple networks
  • All data is stored securely using Hypercore's distributed protocol
  • The node connects to peers via Pear's P2P network

Privacy & Data:

  • Your node ID is generated locally and never shared with third parties
  • Inference requests are processed locally when possible
  • Mining rewards are credited to your configured wallet
  • You can uninstall at any time
