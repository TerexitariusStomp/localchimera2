Resource Markets

Four markets.
One node.

WASI / WASMER

Compute

Sandboxed execution of arbitrary WASM workloads. Jobs run in isolated runtimes with resource quotas enforced at the kernel boundary.

View market
IPFS / HELIA

Storage

Content-addressed peer-to-peer file serving. Pin, serve, and earn on every retrieval via IPFS's CID-based routing mesh.

View market
WEBRTC

Bandwidth

WebRTC relay and routing for real-time data pipelines. Your node becomes a relay in a permissionless mesh network.

View market
QVAC · METAL · VULKAN

Inference

On-device LLM inference with GPU acceleration. Serve local models through the Chimera inference market via QVAC runtime.

Capabilities

One node.
Many jobs.

01

Multi-Resource Mining

BTT AI, Golem, Anyone Protocol, Mysterium, and BTFS. One node, multiple tasking networks running simultaneously.

02

QVAC Runtime

Local task execution with Metal/Vulkan hardware acceleration. GPU-backed inference on your own silicon.

03

Peer-to-Peer

Pear P2P networking with topic-based swarms. Hypercore protocol for append-only distributed data structures.

04

Infrastructure Layers

Docker hardening, Tauri desktop shell, Capacitor mobile sandbox. Runs everywhere your hardware does.

05

Swarm Intelligence

Commander/worker fleet orchestration. Scale your node cluster and coordinate distributed task completion.

06

Open Source

AGPLv3 licensed. Built on QVAC, MLC WebLLM, Hugging Face Transformers, Helia, Wasmer SDK, WebRTC, and Openfort. Fully auditable stack.

Earnings

Your node.
Your earnings.

Devices complete resource tasks in the background across BTT AI, Golem, Anyone Protocol, Mysterium, and BTFS. Resources are paid for via the frontend using Openfort + Request Network; the protocol auto-converts payments to the resource's native token and distributes monthly to your wallet.

◈BTT AI, Golem, Anyone Protocol, Mysterium, and BTFS
◈Openfort + Request Network escrow
◈Openfort wallet + Request Network payments
◈Monthly distributions in each resource's native token
◈Trust score influences job priority
◈Device fingerprinting via signed attestation
